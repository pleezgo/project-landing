Relevance

Modern lifestyle leads to deteriorating public health - poor nutrition, sedentary behavior and insufficient sleep have become widespread problems. Existing solutions either cover only one aspect of health or have limited overall functionality.

This work proposes a comprehensive approach - a single system for tracking nutrition, physical activity, hydration and sleep with detailed statistics and calculations.

Goals & Objectives

The goal of this work is to develop an information system for comprehensive healthy lifestyle support that provides tracking, analysis and visualization of data on nutrition, physical activity, hydration and sleep.

  1. Analyze existing analogues and identify their shortcomings
  2. Design system architecture using React, Node.js and PostgreSQL
  3. Implement nutrition tracking modules with USDA database integration
  4. Implement physical activity, hydration and sleep tracking modules
  5. Develop a dashboard with statistics and BMI and daily nutrient norm calculations

Methodology

The work uses comparative analysis of existing systems, object-oriented design and a component-based approach to interface development. The tech stack includes React and Vite for the frontend, Node.js with Express for the backend, PostgreSQL as the database, JWT for authorization and bcrypt for password security.

Expected Results

  • A working web application with nutrition, activity, hydration and sleep modules
  • Integration with the USDA food database API
  • Automatic BMI and daily nutrient norm calculations based on user profile
  • A unified dashboard with statistics for a selected period
  • Offline mode support
